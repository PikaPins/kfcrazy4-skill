# KFCrazy4 Auto-Post Skill 🍗

WorkBuddy 小红书自动发文 Skill — 每天自动为 KFCrazy4 账号生成疯狂星期四梗文 + AI 配图 + 发布到小红书。

## 安装

```bash
git clone https://github.com/PikaPins/kfcrazy4-skill.git ~/.workbuddy/skills/kfcrazy4-auto-post
```

## 前置依赖

1. 下载 [xiaohongshu-mcp](https://github.com/xpzouying/xiaohongshu-mcp) 并运行在端口 18060
2. 扫码登录小红书

## 使用

在 WorkBuddy 中说「今天发文」即可自动完成：
1. 网上调研最新疯狂星期四梗 → 2. 生成高质量段子 → 3. AI配图 → 4. 发布到小红书

## 文件结构

```
kfcrazy4-auto-post/
├── SKILL.md                        # 完整工作流 SOP
└── references/
    ├── content-style-guide.md      # 内容风格规范 & 配图Prompt模板
    └── research-reference.md       # 全网热门段子调研参考
```
