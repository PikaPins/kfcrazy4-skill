# KFCrazy4 账号内容风格规范

## 账号定位
- **账号名**: KFCrazy4
- **主题**: 100% 围绕「疯狂星期四」梗文化
- **风格**: 发疯文学 / 沙雕 / 抽象 / 互联网梗
- **目标人群**: 年轻打工人、冲浪选手、梗文化爱好者

## 内容基调
- 每次只生成 **1 条段子**
- 核心公式：**引人入胜的故事铺垫 → 情感渲染 → 突然V50反转**
- 语言口语化、细节真实可感
- 故事设定要离谱但有逻辑，越荒谬越好

## 优质段子核心技巧（来自全网热门分析）

### 1. 建立具体语境
快速构建一个具体的场景，让读者立刻代入：
- ❌ 差评： "我是玉皇大帝，要V50"
- ✅ 好评： "跨年还有229天，我们会一直是网友吗……能V我50的加"

### 2. 细节真实可感
加入具体的生活细节增加可信度：
- ❌ 差评： "我穷想吃KFC"
- ✅ 好评： "饿得要死又点了70块钱的外卖……我就是个身高160体重140的无业游民"

### 3. 情感铺垫 + 突然反转
先渲染情绪，再突然转折到V50：
- ❌ 差评： "今天星期四 V我50"
- ✅ 好评： "说了这么多，你都听得到吗？其实我在说今天是肯德基疯狂星期四，v我50，抚慰我支离破碎的心"

### 4. 不同风格模板

#### 模板A：深情铺垫型（最受欢迎）
> [一段深情的、浪漫的、或感人的长篇铺垫]
> [多次反转/情绪积累]
> 其实……今天是疯狂星期四
> V我50 [收尾]

**示例（已验证火爆）**：
> 我的嘴真笨，总能把话题聊死了，跟你找话题好难，何况我这么喜欢你，连口嗨都要挑拣半天呢。我最近越来越期待夜晚了，因为白天都没什么机会能和你说话，只能憋到晚上和你说句晚安。但你可别小看这两个字，它可包含着我今天清晨见到的阳光，中午看到的白云，傍晚遇见的微风，包含着我一天想对你说的话。说了这么多，你都听得到吗？其实我在说今天是肯德基疯狂星期四，v我50，抚慰我支离破碎的心。

#### 模板B：身份扮演型
> [身份/角色] + [具体场景描写]
> [遇到转折/困难 + 细节描写]
> 对方说 / 结果发现是：疯狂星期四 V我50

**示例**：
> 新婚夜，我的丈夫还未与我圆房便领旨出征，临走前向我承诺一定会补偿我。三年后她得胜归来，身边却已多了一个如花似玉的姑娘……V我50听我的复仇计划。

#### 模板C：一本正经胡说八道型
> [伪装成正式文体：通知/药方/欠条/学习总结]
> [大量细节填充，显得很专业]
> 最后引出：V我50

**示例**：
> 本人于2026年5月7日查出两眼昏花饥饿症，所幸有个朋友的姥爷是中医，让我照着这个药方抓药：蛋挞*2 烤翅*3 鸡块*2……谁v我50去抓药。

## 文案结构
- **标题**: 2-3个emoji + 简短标题（不超过 20 字）
- **正文**: 1 条段子，按「情感铺垫 → 反转」结构
- **正文长度**: 控制在 **200-500 字**（但要有内容密度，不能太短）
- **标签**: 选 5-6 个

## 配图 Prompt 构建规范

### 必须包含的元素
1. **画风前缀**: "搞笑漫画风格，"
2. **场景主体**: 段子中的核心角色 + 场景
3. **V50元素**: 必须出现 "V我50" 文字或牌子的描述
4. **KFC元素**: KFC红白条纹 / 炸鸡桶 / KFC包装等
5. **色彩要求**: "色彩鲜艳明快"

### Prompt 模板
```
搞笑漫画风格，[段子的核心场景]，[角色]旁边有"V我50"元素，[KFC元素混搭]，色彩鲜艳明快
```
